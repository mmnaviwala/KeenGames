Elementals version 1.1
------------------------------------------------------------------

	Elementals is a package of particle effects for Unity.
	There are 53 fantastic Shuriken particles separated by 8 kinds of elements
	This package works on all player platforms (Mobile, Web Player, Standalone).
	
	Features:
	
		53 particles from 8 elements.
		- Darkness (2)
		- Earth (3)
		- Fire (28)
		- Ice (4)
		- Light (4)
		- Thunder (5)
		- Water (5)
		- Wind (2)
		
		- Supports all player platforms.
		- Works on both Unity Free/Pro.
